1er Tour

Vous êtes le participant qui a le jeu en main ? Vous devez faire deviner un maximum de mots en un temps limité :
  • Vous pouvez utiliser autant de mot que vous le souhaitez, sauf bien sûr celui à faire deviner
  • Vous n'avez pas d'idées ? Passez au mot suivant, en passant le mot actuel.
  • Lorsque le temps est écoulé, passez l'application à votre voisin de gauche. C'est alors lui qui a le jeu en main: il enchaine avec brio (comme vous).
    Allez, c'est parti !

    • 2ème Tour

    C'est désormais le deuxième tour.
    Vous êtes le participant qui a le jeu en main ? Enchainons sur le même principe, toujours dans un temps limité, avec une consigne que nous ajoutons (vous êtes devenu tellement fort !) :
    Vous ne pouvez utiliser qu'un seul mot pour décrire le mot ! Les autres consignes sont inchangées.

    3ème Tour

    Voici le troisième tour !
    Allons-y sur le même principe, toujours dans un temps limité. Soyons joueur, ajoutons une nouvelle difficulté : vous devez mimer le mot pour le faire deviner.
